Configure Timeline Plot
  • Drift Type: Type of probability distribution to measure the drift of.
  • Number of Attributes per Marginal: Max number of covariate attributes factored in when estimating the different probability distributions
  • Attributes: Covariate attributes to factor in to the estimated probability distributions
  • Group Instances by: Method to determine the grouping of instances into different windows
    • Sequence Key: Back to back instances with the same value for a specified attribute. Useful for dates.
    • Instance Order: Fixed number of instances per window
  • Sequence Keys per Group: Controls size of instance windows depending on value selected for "Group Instances by"
    • Sequence Key: Number of instance groups with different values for the specified attribute in a window
    • Instance Order: Number of instances in each window
  • Grouping Attribute: Attribute used by "Sequence Key" grouping
  • Move Groups Incrementally: When checked, move windows by single instances or by single instance groups depending on Group method selected
